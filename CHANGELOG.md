# Changelog

## 0.4.16

### New Features
- **Live state inspection and editing** via WebSocket bridge for real-time component state management
- **State value components** for displaying and editing typed component state values
- **Emitted context inspection** to view event context data from live components
- **Instance props display** for props of loop-rendered component instances

### Improvements
- Enhanced **component inspector** with separate sections for live state, emitted context, and instance props
- **Component tree** now shows node type badges (layout, runtime), CID counts, and state key indicators
- **Editor integration** with clickable file links to open components directly in your editor
- Improved **bridge connection** status tracking and display across devtools

### Fixes
- Component tree node selection and data passing
